The Schedule Planner Agent is an advanced ReAct-style agent designed to handle complex scheduling tasks. It can automatically create reminders, manage conflicts, and provide a streamlined interface for user productivity. Built with Streamlit and powered by LLMs, it offers a dynamic and interactive scheduling experience.
